board = [["." for _ in range(N)] for _ in range(N)]

def print_board():
    for row in board:
        print(" ".join(row))
    print()

def is_safe(row, col):
    for i in range(N):
        if board[row][i] == 'Q' or board[i][col] == 'Q':
            return False

    # Diagonals
    for i in range(-N, N):
        if 0 <= row + i < N and 0 <= col + i < N:
            if board[row + i][col + i] == 'Q':
                return False

        if 0 <= row + i < N and 0 <= col - i < N:
            if board[row + i][col - i] == 'Q':
                return False

    return True

def computer_move():
    for r in range(N):
        for c in range(N):
            if board[r][c] == "." and is_safe(r, c):
                board[r][c] = "Q"
                print("Computer placed queen at:", r, c)
                return
    print("Computer has no valid move!")

print("8-Queen Interactive Game")
print("Enter row and column (0-7)\n")

while True:
    print_board()

    # User move
    row = int(input("Your Row: "))
    col = int(input("Your Column: "))

    if row < 0 or row >= N or col < 0 or col >= N:
        print("Invalid position!")
        continue

    if board[row][col] == "." and is_safe(row, col):
        board[row][col] = "Q"
    else:
        print("Invalid move!")
        continue

    print_board()

    # Computer move
    computer_move()
    

Explanation

In this experiment, we implemented the 8-Queen problem using Python, which is a classic problem in Artificial Intelligence based on constraint satisfaction. The objective of the problem is to place eight queens on an 8×8 chessboard such that no two queens attack each other horizontally, vertically, or diagonally. We used the is_safe() function to check whether a queen can be placed safely at a given position and the computer_move() function to automatically place a queen in the next valid position. The program also allows user interaction by taking row and column input for queen placement. This practical helped in understanding backtracking, state-space search, and problem-solving techniques used in AI for constraint-based problems.
